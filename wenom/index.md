# WeNoM

::: warning Warning
Der Webnotemanager ist derzeit eine Alpha-Version zum Testen.
:::

Der **WebNotenManager WeNoM** ist eine Online-Plattform zur Erfassung und Verwaltung der Zeugnisnoten, Quartalsnoten, Mahnungen, Teilleistungen, Bemerkungen und Fehlstunden. Er befindet sich derzeit in der Entwicklung (siehe die [Roadmap](../deployment/roadmap.md)) und die Eingabe von Leistungsdaten wie Noten von jedem Internetzugang über einen Browser, etwa von zu Hause aus, unabhängig vom verwendeten Betriebssystem ermöglichen.

::: tip Direkt zur Noteneingabe
Ich möchte nun meine Noten eingeben! Auf zur [Anleitung für Lehrkräfte](./benutzerhandbuch/2_leistungsdaten_lehrkraefte.md)!
:::

Der WebNotenManager bietet Schulen eine effiziente Möglichkeit, die nötigen Leistungsdaten ihrer Schülerinnen und Schüler für den aktuellen Lernabschnitt zu verwalten und ermöglicht Lehrkräften schnell und einfach online auf die benötigten Daten zuzugreifen.

Der WebNotenManager wird parallel zum SVWS-Server und SVWS-Webclient entwickelt, so dass die erfassten Daten einfach mit dem SVWS-Server synchronisiert werden können und anschließend in der Schulverwaltung auf dem SVWS-Server weiterverarbeitet werden können.

Über diesen Bereich der Dokumentation stehen eine *Installationsanleitung*, das *Benutzerhandbuch* und eine Übersicht über die *Geschäftsprozesse* zur Verfügung.
