# Wenom Installation im Webspace Strato

## Voraussetzung

+ Sie haben einen Webspace bei Strato

+ Sie benötigen ein Subdomain

+ Sie benötigen ein Zertifikat

## Subdomain anlegen

Unter "ihre Pakete" z.B. "Wordpress hosting basic" _> Domains -> Domainverwaltung -> Subdomain anlegen

Sie haben zusätzlich eine VM mit ssh Zugriff unter einer fixen IP
-> DNS -> Arecord diese IP Adresse eintragen
