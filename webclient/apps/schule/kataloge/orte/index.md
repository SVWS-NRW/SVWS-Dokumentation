# Orte

Tragen Sie **Orte** ein, mit denen Sie häufig arbeiten.

**Beta**: Das folgende Feature funktioniert derzeit nicht.

Ein größerer Nutzen in der täglichen Arbeit ist jedoch, die Katalogliste so zu sortieren, dass die für Ihre Schule relevante Orte oben in der Liste erscheinen. Nutzen Sie hierfür das Feld **Sortierung**.

![Legen Sie Orte an und sortieren Sie existierende nach oben](./graphics/svws_kataloge_orte.png "Sortieren Sie die Liste so, dass die häufig verwendeten Orte in den Auswahllisten zuerst auftauchen.")

Geben Sie bei Bedarf auch den **Kreis** und das **Bundeslandland** als Zweistellige Kennung ein.
